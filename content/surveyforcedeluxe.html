F.A.Q.

Как создавать опрос?

Сначала необходимо создать категорию опроса. Для этого в SurveyForceDeluxe выберите пункт меню «Категории», нажмите на кнопку «Создать», введите название и описание (поле «Описание» для заполнения необязательно) категории. После этого нажмите на название новой категории, чтобы просмотреть опросы в данной категории. Затем нажмите «Создать» и создайте опрос:

  • Введите название опроса.
  • Введите описание опроса (текст, который будет отображаться на первой странице опроса).
  • Выберите фоновое изображение.
  • Укажите категорию опроса.
  • Выберите язык.
  • Укажите дату окончания.
  • Установите права доступа.

Как устанавливать права доступа к опросу?

Существуют следующие уровни доступа (уровни можно задать при создании/ редактировании опроса):

  • Общий — доступ к опросу имеют все пользователи.
  • Для приглашенных пользователей — доступ к опросу имеют только приглашенные пользователи.
  • Для зарегистрированных пользователей — доступ к опросу имеют только зарегистрированные пользователи.

Я загрузил фоновое изображение. Почему оно не появляется в списке фоновых изображений для опроса?

Чтобы загрузить изображение, нажмите на кнопку рядом с разворачивающимся списком и через появившееся всплывающее окно загрузите изображение. Обновите страницу, через которую Вы создавали или редактировали опрос.

Чтобы загрузить несколько изображений, загрузите их через FTP-соединение в папку images/surveyforce.

Как узнать мнение пользователей о вопросах опроса?

Чтобы получать мнение пользователя о вопросах опроса, создайте для вопросов дополнительную шкалу оценивания. Для этого зайдите в SurveyForceDeluxe → Surveys → Importance Scales, нажмите «Создать», введите текст вопроса (например, «Считаете ли вы вопрос интересным?»), задайте варианты ответов и нажмите «Сохранить». Затем откройте форму для редактирования вопроса и установите для него созданную шкалу оценивания.

Как предложить пользователям оценивать списки, используя указанные шкалы?

Для этого существует вопрос Шкала Лейкерта. Используя шкалу Лейкерта, Вы сможете предлагать пользователю оценивать списки, используя указанную шкалу.

Цвета прямоугольников в вопросе «Перетаскивание» не подходят к установленному шаблону. Как это изменить?

Цвета прямоугольников вопроса «Перетаскивание» можно изменить на странице SurveyForceDeluxe → Configuration.

Почему в пригласительных сообщениях нет ссылки на сайт и опрос?

При вводе текста сообщения на странице SurveyForceDeluxe → Manage e-mails используйте переменную '#link#' — вместо слова link автоматически появится ссылка на опрос.

Как отсортировать результаты на странице Отчеты по ответам пользователей?

Опция «Сортировка по ответам пользователей» становится доступной при сортировке результатов опросов. Из выпадающего списка выберите необходимый опрос. Появится список, через который можно сортировать результаты по ответам.

Как создать ссылку на опрос для зарегистрированных пользователей и гостей сайта?

Зайдите в главное меню и добавьте ссылку на компонент — выберите компонент «Опрос», присвойте имя ссылке, выберите расположение ссылки и нажмите «Сохранить». Откройте SurveyForceDeluxe → Configuration → Menu Manager и добавьте опрос к созданному пункту меню.
