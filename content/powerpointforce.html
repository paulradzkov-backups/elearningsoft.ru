PowerPointForceRU — инструмент для преобразования PowerPoint в SCORM

Вам нужна качественная флеш-презентация, которую можно использовать для решения ежедневных рабочих задач, на сайте или интегрировать в вашу СДО? Тогда наше Windows приложение для Microsoft Office PowerPoint незаменимый инструмент в вашей работе.

Инструмент создания флеш-презентаций PowerPointForceRU позволяет конвертировать PowerPoint слайды в интерактивные веб-презентации и SCORM 1.2/2004 формат. При этом сохраняется вся информация, размещенная на презентационных слайдах: гиперссылки, картинки, диаграммы, анимация, стили и прочее. Кроме того опции этой программы для конвертации PowerPoint слайдов в SCORM и веб-формат позволяют улучшить презентации, сделать их более красивыми.

Использование PowerPointForceRU не требует специальных знаний — JavaScript, HTML, CSS — нужно лишь уметь работать с Microsoft PowerPoint.

Преимущества программы для создания онлайн презентаций PowerPointForceRU

Интуитивно понятный интерфейс позволяет быстро конвертировать PowerPoint слайды в флеш-презентации интерактивного мультимедийного формата. Использование технологий Adobe Flash позволяет сократить размер презентаций и уменьшить время их загрузки пользователями. Созданную презентацию можно просматривать с помощью любого веб-браузера или в системе дистанционного обучения (СДО), поддерживающей работу с форматом SCORM 1.2/2004. Вы можете разместить вашу презентацию в локальной сети (Intranet), на вашем веб-сайте или интегрировать SCORM презентацию в систему дистанционного обучения.

Если ваша организация разрабатывает материалы для систем дистанционного обучения, то у вас будет преимущество, поскольку вы можете создавать файлы формата SCORM. Совместное использование PowerPointForceRU с тестами из QuizForceRU и электронными учебниками из WordForceRU — это легкий и эффективный способ создания качественных курсов/презентаций, которые раньше могли делать только специалисты.

Характеристики инструмента создания флеш-презентаций PowerPointForceRU

  • Интеграция с Microsoft Office как надстройка для Microsoft PowerPoint.
  • Редактирование содержания опубликованной презентации с помощью PowerPointForceRU.
  • Многоязычный интерфейс.
  • Выбор шаблона проигрывателя презентации (3 базовые) и их редактирование.
  • Настройка цветового оформления презентации, использование корпоративного стиля.
  • Выбор вида панели инструментов и возможность изменять текстовые значения элементов панели.
  • Выбор режима демонстрации (во весь экран, в окне).
  • Возможность архивирования презентации в формат .zip.
  • Полная поддержка анимации Microsoft PowerPoint.
  • Возможность поиска по презентации.
  • Публикация в веб-формате для просмотра в любом веб-браузере.
  • Публикация в формате SCORM 1.2/2004 для использования в СДО.

Возможности при создании материалов для СДО:

  • Вставка в презентацию тестов/опросов, созданных в QuizForceRU и передача результатов теста/опроса в СДО.
  • Указание количества набранных баллов в процентном отношении и другие возможности отслеживания завершения тестов/опросов, созданных в QuizForceRU.
  • Вставка в презентацию электронного учебника, созданного при помощи WordForceRU.

Возможности вставки материалов в презентацию:

  • Добавление аудиозаписей в форматах МР3, WAV.
  • Добавление речевого сопровождения к слайдам.
  • Создание списка очередности воспроизведение аудиофайлов для каждого слайда.
  • Вставка видеофайлов формата Flash Movie (.swf) и Flash Film (.flv).
  • Поддержка видео MS PowerPoint.
  • Добавление/редактирование HTML страниц.
  • Вставка информации об авторе презентации (фото, контактные данные).
  • Добавление ссылок и файлов к презентации.
