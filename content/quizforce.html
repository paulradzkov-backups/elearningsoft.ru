QuizForceRU — инструмент для создания тестов и опросов в SCORM формате

QuizForceRU — это программа для создания онлайн тестов и опросов, разработанная как Windows приложение, с применением технологий Adobe Flash. QuizForceRU снабжен многофункциональным и простым в использовании инструментарием для создания флеш тестов, опросов и пулов без необходимости обладания какими-либо техническими знаниями.

С инструментом создания тестов QuizForceRU вы можете создавать учебные флеш тесты для использования их локально на вашем компьютере, для публикации на сайте в Интернет. Также возможна разработка SCORM тестов для интеграции в вашу систему дистанционного обучения — SharePointLMSRU, JoomlaLMSRU, Blackboard и т.п. Создавая качественные и интересные тесты с множеством различных типов вопросов, вы сможете привлечь большее количество студентов на свои курсы.

Возможности QuizForceRU

С помощью QuizForceRU вы сможете создавать тесты и опросы, которые затем можно:

  • Публиковать на вашем сайте, размещать в локальной сети.
  • Публиковать в системе дистанционного обучения, которая поддерживает работу с форматом SCORM 1.2/2004.
  • Использовать как элемент в различных курсах и в составе файлов, созданных в PowerPointForceRU.

Преимущества QuizForceRU

  • Большой набор типов вопросов: заполнение пропусков, множественный выбор / ответ, выбор соответствий, шкала Лайкерта, верно / неверно и другие.
  • Широкие возможности по оформлению тестов с помощью различного цветового и звукового сопровождения.
  • Возможность сохранения результатов посредством СДО или отправки e-mail.
  • Создание сценария прохождения теста / опроса (ответ на все вопросы или пропуск вопросов).
  • Определение действий, которые следует выполнять участнику теста, исходя из набранных баллов.

Стандартные опции программы QuizForceRU

  • Многоязычный интерфейс.
  • Отправка результатов тестов и опросов по e-mail, распечатка.
  • Таймер – ограничение по времени на прохождение теста / опроса.
  • Панель навигации в тесте/опросе для возврата к вопросу, на которые не был еще дан ответ.
  • Определение минимального/максимального количества баллов для прохождения теста.
  • Определение количества попыток ответить на вопрос.
  • Настройка панели инструментов, цветовой гаммы, текста интерфейса теста / опроса.
  • Создание шаблона обратной связи для верных / неверных ответов на вопросы и для всего теста.
  • Использование графических и звуковых файлов, флеш файлов при создании тестов и опросов и другие возможности….
