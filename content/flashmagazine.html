Характеристики:

Зачастую требуется создать журнал на основе PDF файла. И хотя компонент Flash Magazine не осуществляет конвертацию PDF в .jpg и другие графические форматы, мы нашли простое решение данной задачи, которое поможет легко и быстро создавать журналы. Для дополнительной информации, см. Как БЫСТРО создать интерактивный журнал из PDF файла?.

Основные возможности компонента Flash Magazine:

  • Имеется 4 профессиональных шаблона (в них возможно изменять фон, элементы и их цвета, чтобы сделать свой журнал более интересным и индивидуальным).
  • Изменение масштаба страницы (возможно загружать 2 изображения на странице: изображение самой страницы и изображение, которое будет показываться при изменении масштаба).
  • Страницы могут быть изображением, .swf или .flv видео.
  • Опция печати журнала (книги).
  • Обложка журнала (книги).
  • Иконки (быстрая навигация с возможностью предварительного просмотра).
  • Возможность прокрутки слайдов всех станиц.
  • Режим полноэкранного показа.
  • Инструмент одновременной загрузки нескольких изображений.
  • Встроенный WYSIWYG редактор, удобный для форматирования текста.
  • Метаданные (возможны для ключевых слов и описания).
  • Всплывающие окна с описанием журнала или книги в разделе управления журналами, чрезвычайно удобные при большом количестве загруженных журналов и книг.
  • Возможность улучшать или снижать разрешение для журнала (книги).
  • Возможность устанавливать доступ к журналам для отдельных категорий пользователей.

В составе компонента идут также 2 плагина, позволяющие вставлять журналы как при помощи ссылок, так и прямо на страницу с контентом. Для дополнительной информации, см. закладку «Плагины».

Чтобы создать интерактивный журнал или книгу, выполните следующие действия:

Создайте категорию (для нее можно добавить описание) для книг или журналов

Создайте обложку

Создайте страницы путем добавления страниц с изображениями, текстом или видео (вы можете просматривать список всех существующих страниц или отфильтровать их по названию журнала/книги. Видео возможно вставить только на чистую на страницу, без текста и изображений.

Совет: Когда у Вас имеется большое количество страниц, можно использовать кнопку Копировать для создания копий некоторых из них .

Ответы на интересующие Вас вопросы Вы можете получить, воспользовавшись формой обратной связи.

Плагины

В составе компонента идут также 2 плагина: контентный плагин и плагин xtd-кнопка. По завершению установки и публикации контентный плагин позволит вставлять Ваши интерактивные книги и журналы сразу в текст статьи. Например, тэг {flashmagazine id=[mag_id] link=[link]} заменится на ссылку на журнал. Вы также можете вставлять журналы напрямую, без использования ссылок (нажмите на изображение, чтобы его увеличить):

С появлением xtd-кнопки Вам больше не нужно искать ID элемента и добавлять тэги. Вам нужно только нажать на кнопку Flash Magazine в нижнем меню редактора, выбрать книгу или журнал из доступных на сайте и указать, хотите ли добавить ссылку на них (если да, то какую) или поместить прямо в контент. Не забудьте опубликовать плагин после установки.

F.A.Q.

Возможно ли с помощью данного компонента создавать журналы на основе PDF файлов?

К сожалению, невозможно создать подобный скрипт, который работал бы на любом сервере. Все задачи не решить при помощи только лишь одного компонента.

Именно поэтому мы создали компонент "MultiUpload". Он позволяет создавать журналы из пакетов с изображениями. И Вы можете легко и быстро создать новый журнал с заданными параментрами.

Для этого, однако, необходимо использовать какую-либо программу (платную или бесплатную) для конвертации PDF файлов в изображения.

Они очень просты в использовании. Например, нами проверен инструмент 123PDF to Image (http://www.freepdftoimage.com/) для Windows.

Выполните следующие действия:

  • Скачайте и установите 123PDF или любую другую программу для конфертации PDF файлов.
  • Конвертируйте PDF файл в изображения.
  • Загрузите полученные изображения в любую папку на сайте (относящуюся к Joomla) или создайте пакет с изображениями (заметьте, что для загрузки пакета Ваш сервер должен позволять загрузку больших по размеру файлов).
  • Создайте журнал и выберите необходимые настройки.
  • Создайте страницы журнала при помощи опции "MultiUpload".

Как создать журнал Joomla! ?

Выполните следующие действия:

  • Создайте категорию для журнала: нажмите «JoomlaFlashMagazine->Категории», затем нажмите «Создать», введите название и описание категории (если необходимо), и затем нажмите «Сохранить».
  • Выберите «JoomlaFlashMagazine->Журналы» и нажмите «Создать».
  • Введите название, автора и описание для нового журнала.
  • Выберите категорию, языковой файл, шаблон и разрешение.
  • Активируйте/отключите кнопку «Печать».
  • Активируйте/отключите кнопку PDF.
  • Выберите PDF файл.
  • Активируйте/отключите звуки.
  • Выберите фоновый звук.
  • Выберите звук для перелистывания.
  • Активируйте/отключите обложку.
  • Выберите автора изображения, введите его e-mail и биографию (будут показаны, если активирована обложка).
  • Выберите логотип.

Какие типы файлов я могу использовать для загрузки аудио и изображений?

Звуковые файлы могут быть формата mp3 и wav, а изображения - jpg, gif and png. Также можно загружать swf файлы для создания страниц.

Как установить обновления для компонента JoomlaFlashMagazine?

Для начала нужно удалить существующую версию со своего компьютера. Вся информация при этом будут сохранена в базе данных. Затем установите новую версию. Это обновит структуру базы данных, если необходимо и вся информация будет перемещена в новую версию.

Как изменить цвет фона?

Откройте «Управление профилем->Шаблоны» и выберите шаблон, который хотите использовать для журнала. Там имеется опция «Цвет фона», введите код необходимого цвета в пустом поле или просто нажмите кнопку рядом с полем и выберите цвет. Сохраните изменения (нажмите «Сохранитью или «Применить»).

«Цвет элементов» - это цвет всех вспомогательных кнопок на front-end.

«Дополнительный цвет» - это цвет заголовка шаблона, использованного в шаблоне "Бабочка".

Как изменить язык?

Чтобы изменить язык, необходимо заменить файл en-GB.com_flashmagazinedeluxe.ini в папке Language/en-GB/.
