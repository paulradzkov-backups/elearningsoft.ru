Минский филиал МЭСИ o SharePointLMS

В SharePointLMSRU 2.0 нас привлекли инновационные возможности для дистанционного обучения студентов и слушателей. Важным преимуществом является организация работы администраторов системы, преподавателей (тьюторов) и студентов на базе единой платформы...

  1. Использование инфраструктуры Службы каталогов Windows Server позволяет минимизировать затраты на управление пользователями.
  2. Наличие инструментария для подготовки видеоконтента (комментируемых голосом слайдов с навигацией) к существенному снижению затрат на разработку видеоконтента.
  3. Поддержка стандартов SCORM предоставляет возможность обмениваться элементами содержания курсов с другими ВУЗами (в том числе, зарубежными), а также предлагать элементы курсов на рынке контента.
  4. Инструментарий подготовки курсов позволяет обеспечить траекторию обучения для студентов с учетом их индивидуальных особенностей: больше внимания уделять талантливым слушателям, обеспечить дополнительное обучение для отстающих, реализовать право на качественное обучение для лиц с ограниченными возможностями.
  5. Возможности статистического анализа, включенные в рабочие места директора и декана позволяют внести актуальные показатели, касающиеся качества электронного обучения в счетные карты сбалансированной системы показателей. Гибкие возможности шкал оценки работы студентов позволяют интегрировать SharePointLMSRU 2.0 в балльно-рейтинговую систему.
  6. Встроенный инструментарий для проведения видеолекций и видеоконференций приводит к значительной экономии при приобретении специализированного программного обеспечения.
  7. Импортирование документов (тестовых заданий, электронных курсов) из СДО «Прометей» позволяет перенести контент в новую оболочку без потерь.

Вместе с тем, хотелось бы, чтобы в новых версиях были расширены возможности репозитория учебных объектов SharePointLMSRU для управления ими на основе онтологии предметных областей.
